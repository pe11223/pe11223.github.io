1.变量

变量：可以发生改变的一个量，变量是用来区分不同数据的，可以指向一个内存空间，帮我们存储一些数据
变量命名规范：
1.必须是数字或字母或下划线组成
2.不能是数字开头，更不能是纯数字
3.不要用中文
4.不能用python关键字
5.不要太长
6.要有意义
7.推荐下划线和驼峰命名

字符串索引和切片

索引：按照位置提取元素

例如：
s=”我叫XYZ”

我们可以使用索引取出第三个X
print(s[2])

X是在第三个位置，但是我们却输入了二，
因为计算机数数从0开始数

如果我们想要Z那么我们有另外一种方法
print(s[-1])
即可提取到Z

切片：从一个字符串中提取一部分内容。

s=”我叫XYZ，你呢？你叫什么？”

print(s[3:6])

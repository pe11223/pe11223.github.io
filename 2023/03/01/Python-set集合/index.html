set集合

集合是无序的

这样的s并不是一个集合，而是一个字典。
只有在里面添加内容才是一个集合，

打印结果为乱序的
如果我们在集合里添加列表，则会报错
unhashable type:'list'
unhashable:不可哈希
不可哈希：python中的set集合进行数据存储时，需要对数据进行哈希计算，根据计算出来的哈希值进行存储数据
set集合要求存储的数据必须是可以进行哈希计算的
可哈希：不可变的数据类型，int,str,tuple,bool
不可哈希：可变的数据类型：list、dict、set
创建空集合
set()

向集合内添加元素
add()

pop()

remove()

如果我们要进行修改操作呢
只能先删除再添加

实参分类

1.位置参数:按照位置传递参数


2.关键字参数：按名字传参

3.混合参数：位置参数放前面，关键字参数放后边
实参执行时，必须要保障形参有数据

形参分类

1.位置参数：按照位置一个一个声明变量
2.默认值参数：函数声明时给变量一个默认值，如果实参不传递参数，此时默认值生效，否则不生效。
3.动态传参：
Ⅰ：args，表示接收到所有的位置参数的动态传参元组
Ⅱ：**kwargs,表示接收所有的关键字的动态传参字典
顺序：位置>args>默认值>**kwargs

返回值

返回值：函数执行后，会给调用方一个结果，这结果就是返回值
return:
函数只要执行到了return，函数就会立即停止并返回内容，函数内的return后续代码不会执行
1.如果函数内没有return那么外界默认收到None
2.如果写了return
Ⅰ：只写return，后面不跟数据，此时接收到的依然是None
Ⅱ：return值，表示函数有一个返回值，外界能收到一个数据
Ⅲ：return 值1，值2，值3……，此时函数有多个返回值，外界收到的是元组，并且，该元组内存放所有返回值
